# Utilizing data to analyze twitter NBA player 

We managed to identify the **people, objective, process, data** framework for NBA player as follows:

## People:
 - Coach
 - Viewer
 - Influencers
 - Athletes
 
## Objective:
 - Sell tickets
 - Talent scouting
 - Optimize training
 - Sell brand merchandise
 
## Process:
 - Pick more interesting players
 - Use different training techniques
 
## Data:
 - Performance of each player
 - Endorsement for each team
 - Correlation between various physical attributes to performance
